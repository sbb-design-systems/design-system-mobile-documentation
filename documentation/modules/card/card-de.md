## 1. Was macht das Element?
* Gruppiert Funktionen/Informationen für einen schnellen Zugriff auf diese.


## 2. Wann soll das Element eingesetzt werden?
* Zur Sammlung/Strukturierung von Informationen zu einem Thema.
* Wenn mindestens eine Interaktion mit dem Modul möglich ist (z.B. Details aufrufen).


## 3. Regeln
* Optional kann eine zusätzliche "Call to Action" unterhalb der Card angezeigt werden
* Der Titel ist optional
* Es können mehrere Cards nacheinander dargestellt werden (Horizontal Scrolling)
* Der Schatten ist optional und sollte nur eingesetzt werden, wenn das Modul über einem Element dargestellt wird.
